# TechTrove - Bharat's Premier Unified Service Network

A modern, high-performance web ecosystem representing TroveTech's nationwide service infrastructure in India.

## Features
- **Modern UI**: Sleek, cinematic design with glassmorphism and smooth animations.
- **India-Centric**: Tailored content and realistic professional imagery for the Indian market.
- **Responsive**: Fully optimized for desktop and mobile devices.
- **Vercel Ready**: Optimized with `vercel.json` for clean URLs and high security.

## Local Development
To run this project locally:
1. Open a terminal in this directory.
2. Run: `npm start`
3. View the site at `http://localhost:3000`.

## Deployment
This site is optimized for one-click deployment to Vercel. 
- **Method 1**: Use the Vercel CLI by running `vercel` in this directory.
- **Method 2**: Connect this folder to a GitHub repository and link it to Vercel for automatic CI/CD.
